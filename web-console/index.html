❓ FAQ

Часто задаваемые вопросы

Как добавить бота на сервер?

Перейдите в раздел Dashboard и нажмите "Invite Bot". Вы будете перенаправлены на страницу Discord где сможете выбрать сервер для добавления бота.

Как изменить префикс команд?

Откройте Dashboard → Bot Settings → Command Prefix. Введите новый префикс и нажмите "Save Changes".

Почему бот не отвечает на команды?

Проверьте следующее:

  • Бот онлайн (зелёный индикатор в Dashboard)
  • У бота есть права на чтение/отправку сообщений
  • Вы используете правильный префикс
  • Канал не в списке исключений
Как посмотреть логи действий?

Перейдите в раздел Logs в боковом меню. Там доступны логи сообщений и действий модерации.

Можно ли добавить бота на несколько серверов?

Да! Бот может работать на нескольких серверах одновременно. Используйте "Server Folders" для организации серверов по группам.

Переменные — хранилища данных

Переменная — это коробка с именем, в которую можно положить данные.

# Создание переменных
name = "Инженер"          # Строка (текст)
age = 25                   # Целое число
height = 1.75              # Дробное число
is_engineer = True         # Логический тип (True/False)

# Использование
print("Меня зовут", name)
print("Мне", age, "лет")
print("Мой рост:", height, "м")

Правила именования переменных:

  • ✅ user_name, age, total_sum — хорошо
  • ❌ 1name — нельзя начинать с цифры
  • ❌ user-name — нельзя дефис, только подчёркивание
  • ❌ class, if, for — зарезервированные слова

💡 Совет: Называй переменные понятно! x — плохо, user_age — хорошо.

— Почему программисты путают Хэллоуин и Рождество?
— Потому что Oct 31 = Dec 25! (31 в восьмеричной = 25 в десятичной) 🎃🎄

Операторы

Арифметические операторы:

a = 10
b = 3

print(a + b)   # 13 — сложение
print(a - b)   # 7  — вычитание
print(a * b)   # 30 — умножение
print(a / b)   # 3.333... — деление
print(a // b)  # 3  — целочисленное деление
print(a % b)   # 1  — остаток от деления
print(a ** b)  # 1000 — возведение в степень

Операторы сравнения:

x = 5
y = 10

print(x == y)   # False — равно
print(x != y)   # True  — не равно
print(x < y)    # True  — меньше
print(x > y)    # False — больше
print(x <= y)   # True  — меньше или равно
print(x >= y)   # False — больше или равно

Логические операторы:

a = True
b = False

print(a and b)  # False — И (оба должны быть True)
print(a or b)   # True  — ИЛИ (хотя бы один True)
print(not a)    # False — НЕ (инверсия)

Строки — работа с текстом

# Создание строк
name = "Python"
greeting = 'Привет!'
multiline = """Это
многострочный
текст"""

# Конкатенация (склеивание)
full = "Hello, " + name + "!"

print(word[0])   # "P" — первый символ
print(word[-1])  # "n" — последний символ
print(word[0:3]) # "Pyt" — срез (с 0 до 3, не включая 3)

💡 Совет: Используй f-строки! Они читаемее и быстрее.

Условия — if/elif/else

Условия позволяют выполнять разный код в зависимости от ситуации.

# Простое условие
age = 18

if age >= 18:
    print("Ты взрослый!")
else:
    print("Ты ещё ребёнок")

# Несколько условий
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Твоя оценка: {grade}")

# Вложенные условия
is_member = True
has_ticket = True

if is_member:
    if has_ticket:
        print("Добро пожаловать!")
    else:
        print("Нужен билет")
else:
    print("Только для членов клуба")
⚠️ ВАЖНО: В Python отступы (4 пробела) — это часть синтаксиса! Без них код не работает.
🔄

Циклы — повторение действий

Цикл for — проходит по каждому элементу:

# Перебор списка
fruits = ["яблоко", "банан", "апельсин"]
for fruit in fruits:
    print(f"Я люблю {fruit}")

# range() — генерирует числа
for i in range(5):         # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):      # 1, 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8 (шаг 2)
    print(i)

Цикл while — пока условие истинно:

# Простой while
count = 0
while count < 5:
    print(count)
    count += 1  # ВАЖНО: не забудь изменять условие!

# Бесконечный цикл с break
while True:
    answer = input("Выйти? (да/нет): ")
    if answer == "да":
        break  # Выход из цикла
    print("Продолжаем...")

continue — пропустить итерацию:

for i in range(10):
    if i == 5:
        continue  # Пропустить 5
    print(i)  # Напечатает 0,1,2,3,4,6,7,8,9

— Как программист находит выход из бесконечного цикла?
— Ctrl+C! Или идёт за кофе и ждёт пока память закончится ☕💀

Списки (list)

Список — упорядоченная коллекция элементов.

# Создание списка
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]  # Можно разные типы
empty = []

# Доступ к элементам
print(numbers[0])   # 1 — первый элемент
print(numbers[-1])  # 5 — последний элемент
print(numbers[1:3]) # [2, 3] — срез

# Изменение
numbers[0] = 100
print(numbers)  # [100, 2, 3, 4, 5]

# Добавление элементов
numbers.append(6)      # В конец
numbers.insert(0, 0)   # В начало (индекс 0)

# Удаление элементов
numbers.remove(100)    # Удалить по значению
del numbers[0]         # Удалить по индексу
last = numbers.pop()   # Удалить последний и вернуть

# Полезные методы
print(len(numbers))    # Длина списка
print(sum(numbers))    # Сумма (для чисел)
print(min(numbers))    # Минимум
print(max(numbers))    # Максимум
numbers.sort()         # Сортировка
numbers.reverse()      # Переворот

# Проверка наличия
if 3 in numbers:
    print("3 есть в списке!")

List Comprehension — мощный способ создания списков:

# Вместо:
squares = []
for x in range(10):
    squares.append(x ** 2)

# Можно написать:
squares = [x ** 2 for x in range(10)]
print(squares)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# С условием

Словари (dict)

Словарь — коллекция пар "ключ: значение".

# Создание словаря
user = {
    "name": "Инженер",
    "age": 25,
    "is_active": True,
    "skills": ["Python", "сантехника", "электрика"]
}

# Доступ к значениям
print(user["name"])        # "Инженер"
print(user.get("name"))    # "Инженер" (безопаснее!)
print(user.get("job", "Не указано"))  # "Не указано" (если нет ключа)

# Изменение/добавление
user["age"] = 26           # Изменить
user["email"] = "a@b.com"  # Добавить новый ключ

# Удаление
del user["email"]
user.pop("is_active")

# Перебор
for key in user:
    print(f"{key}: {user[key]}")

# Или так:
for key, value in user.items():
    print(f"{key} = {value}")

# Проверка ключа
if "name" in user:
    print("Ключ 'name' существует!")

💡 Совет: Используй .get() вместо [] — не будет ошибки если ключа нет!

Функции

Функция — это переиспользуемый блок кода с именем.

# Простая функция
def greet():
    print("Привет!")

greet()  # Вызов функции

# Функция с параметрами
def greet_user(name):
    print(f"Привет, {name}!")

greet_user("Инженер")  # Привет, Инженер!

# Функция с возвратом значения
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

# Параметры по умолчанию
def greet(name, greeting="Привет"):
    return f"{greeting}, {name}!"

print(greet("Вася"))              # Привет, Вася!
print(greet("Вася", "Здоро́во"))   # Здоро́во, Вася!

# Произвольное количество аргументов
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3, 4, 5))  # 15

# Именованные аргументы
def create_user(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

create_user(name="Вася", age=25, city="Москва")

Обработка ошибок (try/except)

Ошибки — это нормально! Главное — уметь их обрабатывать.

# Без обработки — программа упадёт
# print(10 / 0)  # ZeroDivisionError

# С обработкой
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Ошибка: деление на ноль!")

# Несколько типов ошибок
try:
    number = int(input("Введи число: "))
    result = 100 / number
    print(f"Результат: {result}")
except ValueError:
    print("Это не число!")
except ZeroDivisionError:
    print("Нельзя делить на ноль!")
except Exception as e:
    print(f"Неизвестная ошибка: {e}")
finally:
    print("Этот код выполнится ВСЕГДА")

# Поднятие ошибки
def check_age(age):
    if age < 0:
        raise ValueError("Возраст не может быть отрицательным!")
    return age

💡 Совет: Не используй голый except: — всегда указывай тип ошибки!

Работа с файлами

# Запись в файл
with open("test.txt", "w", encoding="utf-8") as file:
    file.write("Привет, мир!\n")
    file.write("Это вторая строка")

# Чтение файла
with open("test.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)

# Чтение построчно
with open("test.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())  # strip() убирает \n

# Дозапись (не перезаписывает!)
with open("test.txt", "a", encoding="utf-8") as file:
    file.write("\nЭто новая строка")

# Работа с JSON
import json

data = {"name": "Python", "version": 3.12}

# Запись JSON
with open("data.json", "w", encoding="utf-8") as file:
    json.dump(data, file, ensure_ascii=False, indent=2)

# Чтение JSON
with open("data.json", "r", encoding="utf-8") as file:
    loaded = json.load(file)
    print(loaded["name"])

Режимы открытия:

  • "r" — чтение (файл должен существовать)
  • "w" — запись (перезаписывает!)
  • "a" — дозапись (добавляет в конец)
  • "x" — создание (ошибка если файл есть)
⚠️ Всегда используй with — файл автоматически закроется!

ООП — Классы и объекты

ООП — способ организации кода через объекты.

# Определение класса
class Dog:
    # Конструктор — вызывается при создании объекта
    def __init__(self, name, age):
        self.name = name  # Атрибут
        self.age = age
    
    # Метод (функция класса)
    def bark(self):
        print(f"{self.name} говорит: Гав!")
    
    def info(self):
        return f"{self.name}, {self.age} лет"

# Создание объектов
dog1 = Dog("Рекс", 3)
dog2 = Dog("Бобик", 5)

# Использование
print(dog1.name)   # Рекс
dog1.bark()        # Рекс говорит: Гав!
print(dog2.info()) # Бобик, 5 лет

# Наследование
class Puppy(Dog):
    def __init__(self, name, age, toy):
        super().__init__(name, age)  # Вызов родительского конструктора
        self.toy = toy
    
    def play(self):
        print(f"{self.name} играет с {self.toy}")

puppy = Puppy("Шарик", 1, "мячиком")
puppy.bark()  # Шарик говорит: Гав!
puppy.play()  # Шарик играет с мячиком

💡 Совет: Используй классы когда у тебя есть "сущности" с данными и поведением.

Сантехника

🔧 Течёт кран (капает)

"Сынок, это проще чем турель!"

Причина: Износилась резиновая прокладка внутри головки крана.

Инструменты: Разводной ключ, отвёртка, новая прокладка

  1. Перекрой воду — вентили под раковиной (их два: горячая и холодная). Поверни по часовой стрелке до упора
  2. Открути заглушку — обычно сверху на ручке есть декоративная крышечка (подденьте ножом)
  3. Открути винт — он под крышечкой, держит ручку крана
  4. Сними ручку — потяни вверх
  5. Открути головку — разводным ключом (против часовой)
  6. Достань старую прокладку — она внизу головки, чёрная резинка
  7. Вставь новую — такого же размера (возьми старую в магазин как образец!)
  8. Собери всё обратно — в обратном порядке
  9. Открой воду — проверь, не течёт ли

💡 Лайфхак: Сфотографируй каждый этап — не забудешь как собрать!

💰 Прокладки: 50-200₽ (набор) ⚠️ Если кран шаровый (рычаг вместо барашка) — там картридж, его меняют целиком (500-1500₽)

🚽 Бачок не набирает воду

"Эта штука важнее телепорта!"

Причина: Либо засорился клапан, либо сломался механизм.

Инструменты: Руки, возможно — плоскогубцы

  1. Сними крышку бачка — обычно просто лежит сверху, иногда закреплена кнопкой (открути)
  2. Осмотри механизм:
    • Поплавок — пластиковый шар или цилиндр на рычаге. Должен свободно двигаться вверх-вниз
    • Клапан подачи — откуда идёт вода. Проверь, не забит ли ржавчиной
    • Сливной механизм — большая штука в центре
  3. Если поплавок застрял — пошевели его, почисти от накипи
  4. Если клапан не пропускает воду — перекрой воду, разбери клапан, почисти сеточку
  5. Если ничего не помогает — замени арматуру целиком (продаётся готовым комплектом)

💡 Лайфхак: Сфоткай старую арматуру и покажи в магазине — подберут совместимую!

💰 Арматура: 500-1500₽, ремкомплект: 100-300₽

🚰 Засорилась раковина

"Засор — это просто ленивая вода!"

Причина: Волосы, жир, мыло скопились в сифоне или трубе.

Инструменты: Вантуз, трос, средство для прочистки

Способ 1 — Вантуз:

  1. Закрой перелив тряпкой (дырка сбоку раковины)
  2. Налей воды чтобы покрыла чашку вантуза
  3. Прижми плотно и делай резкие движения вверх-вниз 10-15 раз

Способ 2 — Сифон:

  1. Поставь ведро под сифон
  2. Открути нижнюю часть сифона (руками или ключом)
  3. Вычисти грязь, промой
  4. Собери обратно

Способ 3 — Химия:

  1. Крот, Tiret — залей, подожди 15-30 минут, промой горячей водой
💰 Вантуз: 100-300₽, Трос: 200-500₽, Крот: 50-150₽ ⚠️ НЕ СМЕШИВАЙ разные химические средства — опасно!

Электрика

⚡ Выбивает автомат

"Осторожней, напарник! Электричество шуток не понимает!"

Причина: Короткое замыкание, перегрузка сети или неисправный прибор.

Диагностика:

  1. Отключи ВСЕ приборы из розеток (холодильник, микроволновка, обогреватель — ВСЁ)
  2. Включи автомат — подними рычаг вверх
  3. Если сразу выбило — проблема в проводке. ВЫЗОВИ ЭЛЕКТРИКА!
  4. Если держится — подключай приборы по одному с интервалом 1 минуту
  5. Виновник — тот прибор, после которого снова выбьет

Частые причины:

  • Обогреватель + чайник одновременно = перегрузка
  • Старый удлинитель с оплавленными контактами
  • Неисправный прибор (часто — старый холодильник или стиралка)

💡 Совет: Если автомат выбивает регулярно — замени на более мощный (только если проводка позволяет!) или раздели нагрузку.

⚠️ НИКОГДА не ставь "жучок" вместо автомата — это пожар!

🔌 Не работает розетка

"Розетка — это как телепорт для электронов!"

Причина: Ослабли контакты, отгорел провод, или проблема не в розетке.

Инструменты: Индикаторная отвёртка, обычная отвёртка

  1. СНАЧАЛА — проверь, есть ли свет в комнате. Если нет — проблема в автомате или во всей линии
  2. Проверь другие розетки — может, выбила группа
  3. ОТКЛЮЧИ АВТОМАТ — это обязательно!
  4. Сними крышку розетки — обычно 1-2 винта
  5. Осмотри провода:
    • Чёрные, оплавленные места = отгорел контакт
    • Провод выскочил из клеммы = вставь и затяни
    • Всё в порядке = проблема глубже в проводке
  6. Подтяни клеммы — часто контакты просто ослабли со временем
💰 Новая розетка: 100-500₽ ⚠️ Если провод оплавился или повреждена изоляция — ВЫЗОВИ ЭЛЕКТРИКА!

💡 Не горит лампочка

"Да будет свет! Или не будет?"

Шаг 1: Замени лампочку. Серьёзно, в 80% случаев просто перегорела.

Шаг 2: Если новая не горит — проверь:

  • Выключатель — щёлкает? Контакты не оплавились?
  • Патрон — не окислились ли контакты внутри
  • Автомат — может выбил

Чистка патрона:

  1. Отключи автомат!
  2. Выкрути лампочку
  3. Почисти контакты в патроне наждачкой или ножом
  4. Если пластиковый патрон оплавился — замени люстру или вызови мастера
💰 Лампочка LED: 100-300₽, патрон: 50-150₽

💡 Совет: Переходи на LED — они не греются, служат годами, экономят электричество!

⚡ Искрит розетка

"ОПАСНОСТЬ! Это срочно!"

Причина: Ослабли контакты, повреждена проводка, или перегрузка.

  1. НЕ ПОЛЬЗУЙСЯ этой розеткой!
  2. Отключи автомат на эту линию
  3. Осмотри — если видно оплавленный пластик или чёрные следы — ЗАМЕНА
  4. Если просто ослабли контакты — подтяни клеммы (автомат ВЫКЛЮЧЕН!)
⚠️ Искрящая розетка = риск пожара. Не откладывай ремонт!

  2. Купи средство для декальцинации (или разведи лимонную кислоту 30г/литр)
  3. Залей, запусти цикл, оставь на 15 минут
  4. Слей, промой 3 цикла чистой водой

💡 Делай каждые 2-3 месяца!

💰 Средство: 200-500₽, лимонная кислота: 50₽

🧊 Холодильник плохо морозит

"Терморегуляция — это серьёзно!"

  1. Проверь резинку двери — приложи бумагу, закрой дверь. Если вытаскивается легко — меняй резинку
  2. Почисти конденсатор сзади от пыли (пылесосом)
  3. Не ставь горячее внутрь
  4. Проверь вентилятор — должен крутиться
⚠️ Если не помогло и есть иней на задней стенке — это утечка фреона. ВЫЗОВИ МАСТЕРА!

🍞 Микроволновка искрит

"ОПАСНОСТЬ! Но часто легко чинится."

  1. Слюдяная пластина — прогорела (справа внутри)
  2. Купи новую (универсальную), вырежи по форме, замени
  3. Или закажи оригинальную под модель
💰 Слюдяная пластина: 100-300₽ ⚠️ Перед ремонтом ОТКЛЮЧИ ОТ СЕТИ и подожди 5 минут!
🚗

Автомобиль (базовое)

🔋 Сел аккумулятор

"Как турель без Sentry Gun!"

Прикуривание от другой машины:

  1. Красный провод → на ПЛЮС обоих аккумуляторов
  2. Чёрный → на МИНУС донора, потом на МЕТАЛЛ твоего авто (не на аккум!)
  3. Заведи донора, подожди 2 мин, заведи свою
  4. Отключай в обратном порядке
⚠️ Не перепутай полярность — сгорит электроника! 💰 Провода для прикуривания: 500-1500₽

🛞 Как заменить колесо

"Базовый навык каждого водителя!"

  1. Поставь машину на ровное место, включи ручник
  2. Ослабь болты (НЕ СНИМАЙ!) пока колесо на земле
  3. Подними домкратом (под порог, где усилитель)
  4. Сними колесо, поставь запаску
  5. Закрути болты крест-накрест
  6. Опусти, дотяни болты

💡 Лайфхак: Домкрат ставь на ровную доску если грунт мягкий!

💨 Заправка кондиционера

"Не мёрзни, напарник!"

  1. Купи баллон с фреоном R134a со шлангом
  2. Найди порт низкого давления (синяя крышка)
  3. Включи кондей на максимум
  4. Подключи шланг, заправляй пока давление не станет нормальным (по манометру)
💰 Баллон с фреоном: 500-1000₽ ⚠️ Если кондей дует тёплым сразу после заправки — утечка! В сервис.

❓ FAQ

Часто задаваемые вопросы

🤖 Как подключить бота к серверу?

Перейдите по invite-ссылке бота и выберите сервер для подключения. Бот автоматически запросит необходимые права.

⚙️ Как изменить prefix команд?

Откройте Dashboard → Bot Settings → Command Prefix. Изменения сохраняются автоматически.

📊 Где посмотреть статистику?

Раздел Analytics показывает графики активности, сообщений и топ пользователей вашего сервера.

🎮 Как играть в мини-игры?

Выберите Tetris или Snake в меню "Развлечение". Управление: стрелки, пробел, ESC для паузы.

🔧 Как использовать 3D Конструктор?

Maya-like интерфейс: Q/W/E/R для выбора инструмента (Select/Move/Rotate/Scale). Добавляйте примитивы из панели справа.
